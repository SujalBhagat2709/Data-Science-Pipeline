## ğŸ”¹ Step-by-Step Workflow

### 1. Data Collection  
**File:** `01_data_collection.py`

- Loads raw data from a CSV file  
- Displays basic dataset information  
- Saves a snapshot of the raw data  

**Goal:** Obtain and understand the raw data.

---

### 2. Data Cleaning (Preprocessing)  
**File:** `02_data_cleaning.py`

- Removes duplicate records  
- Handles missing numeric values using median  
- Handles missing categorical values  
- Saves cleaned data  

**Goal:** Ensure data quality and reliability.

---

### 3. Data Transformation  
**File:** `03_data_transformation.py`

- Scales numeric features  
- Converts data types where required  
- Saves transformed data  

**Goal:** Prepare data for analysis and modeling.

---

### 4. Exploratory Data Analysis (EDA)  
**File:** `04_eda.py`

- Generates statistical summaries  
- Plots correlation heatmaps  
- Visualizes feature distributions  

**Goal:** Understand patterns, trends, and relationships in data.

---

### 5. Data Analysis  
**File:** `05_data_analysis.py`

- Computes mean, median, and variance  
- Analyzes relationships between variables  
- Identifies key insights  

**Goal:** Extract meaningful insights from the data.

---

### 6. Visualization & Reporting  
**File:** `06_visualization_reporting.py`

- Creates charts and graphs  
- Generates summary reports  
- Visualizes department-wise comparisons  

**Goal:** Communicate insights clearly and effectively.

---

### 7. Decision Making  
**File:** `07_decision_making.py`

- Interprets analysis results  
- Generates actionable insights  
- Prepares data for machine learning pipelines  

**Goal:** Support informed, data-driven decision making.
